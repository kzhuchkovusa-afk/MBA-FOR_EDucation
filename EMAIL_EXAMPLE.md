# Email Notification Example

When a user submits the application form, Netlify will send an email to **info@gocoding.tech** with the following information:

---

## Sample Email Content

**Subject:** New form submission: apply

**From:** Netlify Forms <no-reply@netlify.com>

**To:** info@gocoding.tech

---

### Form Submission Details

**Form Name:** apply

**Submission Date:** 2026-01-13 18:30:45 UTC

---

### Application Data

**Full Name:** John Smith

**Email:** john.smith@example.com

**Role:** Owner

**Business Type:** K-12 private school

**Grades Served:** K-8

**Current Enrollment Range (# students):** 51-100 students

**Biggest Bottleneck:** Enrollment

**Goal for next 90 days:**
We need to increase enrollment by 20% and streamline our admissions process. Currently spending too much time on manual follow-ups with parents and losing leads. Want to implement automated systems and improve conversion from inquiry to enrollment.

---

### Actions Required

1. Review application details
2. If qualified, send Calendly booking link (or applicant already booked via thank-you page)
3. Conduct 15-minute clarity call
4. If approved, send membership access credentials

---

## Netlify Forms Dashboard

You can also view all submissions in Netlify dashboard:

1. Log in to Netlify
2. Select your site
3. Go to **Forms** tab
4. View all submissions with timestamps
5. Export as CSV if needed

---

## Email Notification Settings

To modify email notifications:

1. Netlify Dashboard → Site Settings → Forms
2. Click **Notifications**
3. Edit email notification settings
4. Can add multiple recipients
5. Can customize email format (plain text / HTML)

---

**Note:** Make sure to check spam folder if emails are not arriving in inbox.
